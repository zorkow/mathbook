Example 18.17.

Every PID is a UFD, but it is not the case that every UFD is a PID. In CorollaryÂ 18.31, we will prove that \({\mathbb Z}[x]\) is a UFD. However, \({\mathbb Z}[x]\) is not a PID. Let \(I = \{ 5 f(x) + x g(x) : f(x), g(x) \in {\mathbb Z}[x] \}\text{.}\) We can easily show that \(I\) is an ideal of \({\mathbb Z}[x]\text{.}\) Suppose that \(I = \langle p(x) \rangle\text{.}\) Since \(5 \in I\text{,}\) \(5 = f(x) p(x)\text{.}\) In this case \(p(x) = p\) must be a constant. Since \(x \in I\text{,}\) \(x = p g(x)\text{;}\) consequently, \(p = \pm 1\text{.}\) However, it follows from this fact that \(\langle p(x) \rangle = {\mathbb Z}[x]\text{.}\) But this would mean that \(3\) is in \(I\text{.}\) Therefore, we can write \(3 = 5 f(x) + x g(x)\) for some \(f(x)\) and \(g(x)\) in \({\mathbb Z}[x]\text{.}\) Examining the constant term of this polynomial, we see that \(3 = 5 f(x)\text{,}\) which is impossible.
