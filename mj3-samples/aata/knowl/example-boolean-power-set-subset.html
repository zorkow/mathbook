Example 19.2.

Let \(X\) be any set. We will define the power set of \(X\) to be the set of all subsets of \(X\text{.}\) We denote the power set of \(X\) by \({\mathcal P}(X)\text{.}\) For example, let \(X = \{ a, b, c \}\text{.}\) Then \({\mathcal P}(X)\) is the set of all subsets of the set \(\{ a, b, c \}\text{:}\)

\begin{align*} & \emptyset & & \{ a \} & & \{ b \} & & \{ c \} &\\ & \{ a, b \} & & \{ a, c\} & &\{ b, c\} & & \{ a, b, c \}. & \end{align*}
