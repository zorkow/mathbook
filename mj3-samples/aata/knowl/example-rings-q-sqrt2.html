Example 16.14.

The set \({\mathbb Q}( \sqrt{2}\, ) = \{ a + b \sqrt{2} : a, b \in {\mathbb Q} \}\) is a field. The inverse of an element \(a + b \sqrt{2}\) in \({\mathbb Q}( \sqrt{2}\, )\) is

\begin{equation*} \frac{a}{a^2 - 2 b^2} +\frac{- b}{ a^2 - 2 b^2} \sqrt{2}\text{.} \end{equation*}
