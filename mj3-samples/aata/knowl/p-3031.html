A set \(B\) is a Boolean algebra if and only if there exist binary operations \(\vee\) and \(\wedge\) on \(B\) satisfying the following axioms.

  1. \(a \vee b = b \vee a\) and \(a \wedge b = b \wedge a\) for \(a, b \in B\text{.}\)

  2. \(a \vee ( b \vee c) = (a \vee b) \vee c\) and \(a \wedge ( b \wedge c) = (a \wedge b) \wedge c\) for \(a, b, c \in B\text{.}\)

  3. \(a \wedge ( b \vee c ) = (a \wedge b ) \vee ( a \wedge c )\) and \(a \vee ( b \wedge c ) = (a \vee b ) \wedge ( a \vee c )\) for \(a, b, c \in B\text{.}\)

  4. There exist elements \(I\) and \(O\) such that \(a \vee O = a\) and \(a \wedge I = a\) for all \(a \in B\text{.}\)

  5. For every \(a \in B\) there exists an \(a' \in B\) such that \(a \vee a' = I\) and \(a \wedge a' = O\text{.}\)
